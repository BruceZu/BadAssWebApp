BadAssWebApp
============

Java WebApp demo featuring jQuery, JSON, Servlet 3.0, Spring 3.1, JAXB 2.2.5, JDO 3.0 & DataNucleus 3.0

Install:
* JBoss AS7 - http://www.jboss.org/as7 - 7.1.1.Final
* MySQL Community Server - http://www.mysql.com/downloads/mysql/ - 5.5.23
* mvn clean install
* Copy target/BadAssWebApp-1.0.war to jboss deploy dir
* Open http://localhost:8080/BadAssWebApp-1.0/ in brows
